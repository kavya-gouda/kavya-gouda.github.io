# Welcome

This is your first blog post.

## How to add another post

1. Create a new markdown file under `posts/<slug>/post.md` (for example: `posts/my-second-post/post.md`)
2. Write your title as the first markdown heading (e.g. `# My Second Post`)

Your `index.html` will list posts from `posts.json`, and `post.html` will render the markdown.
